## 瀏覽此專案

### 使用 Docker 
1. ``git pull`` 此專案 ( 必要資源: docker-compose.yaml、/sql )
2. 執行 ``docker compose up`` 
3. 訪問 [localhost:3000](http://localhost:3000) 進行瀏覽
